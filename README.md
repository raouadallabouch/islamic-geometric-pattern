Islamic Star Pattern Ripple

An interactive visualization of the classic Islamic Star Pattern, featuring an aesthetic mouse ripple effect that distorts the geometry in real-time.

This is not a static image, but a mathematical simulation. It uses Hankin's "Polygons in Contact" algorithm to draw the star pattern procedurally. The intersection points are treated as physical nodes suspended in a grid; when you move your mouse, you push these nodes, creating the fluid, organic ripple effect over the rigid geometric structure.
